# glambook (event build)

## Known issues (6)

**Bug 1 — Home navigation**  
After you search, open a salon, then close the booking window and click **Home** in the header or footer, the page does not go back to the top. You have to **reload the site** to see the home area again.

**Bug 2 — Phone on booking**  
The booking form still shows the phone field as **required** (*), but the form **accepts a booking with an empty phone number**.

**Bug 3 — One service per booking**  
The appointment form only lets you **pick one service** in a single dropdown. There is **no way to choose multiple services** in one booking.

**Bug 4 — Duplicate salon sign-up**  
The same shop can be **registered more than once** (same name or details). Nothing stops duplicate registrations.

**Bug 5 — Salon contact number**  
The salon registration **phone/contact field** is not checked for real phone rules; it can **save letters and other non-numeric text** as the number.

**Bug 6 — No delete for a added shop**  
You can add a new salon, but the app has **no button or way to remove or delete** a shop from the list.
